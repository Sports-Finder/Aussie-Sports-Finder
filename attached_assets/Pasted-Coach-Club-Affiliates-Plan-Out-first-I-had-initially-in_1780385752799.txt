Coach & Club Affiliates (Plan Out first)

I had initially introduced a rule where Coaches and Players cannot connect, but now thinking about it, to erase the hassle of coaches having to  approach the club to advertise for players wanted & trials info, I think we should put the following Coach account system in place;

- Change the Create an Account tab from 'I am a Coach looking for a Club' to 'I am a Coach Looking for a Club or Players'
- The Coach Account Set up is exactly the same.
- When they go to Post an Advert, the only advert type option they initially have is the 'Coach looking for a Team or Club' button. However they do see the 'Players Wanted' and 'Trials Information' advert type buttons (that Club accounts see) but these are greyed out/not selectable. If they try to select it, a pop up displays the following text 'You cannot post Players Wanted or Trials Info as you are currently not affiliated with a club. Ask your club to make you an Affiliated Coach. If your Club doesn't have an account, they will need to create one first.' and an OK button to dismiss.
- Once a club account is set up and approved, under Club Profile box, include a 2nd box that is labelled COACH AFFILIATES with a view/edit button, which opens a new [CLUB NAME] COACH AFFILATES Page. There is a back to Club Profile option at the top left.
	- On this Coach Affiliates page, the club is able to view a list of Coaches that they have agreed to affiliate 	with. These are displayed as profile pic, Coach Name and the team name and age group (which is a new field Club's 	set up under the Edit option. They can open up their profile page and view this in full (once a coach accepts 	their request). They can also choose to remove a Coach. By doing this they are warned that any adverts or 	chats with this current affiliation will be removed. It is important not remove any of this coaches 	personal 	"Looking for a club advert types' if they have one.	
	- A Club can also request to add a Coach as an Affiliate to their club. There is an 'Add a Coach Affiliate' 	button at the top of the page.  This opens a pop up box that has a Full Name search bar. The search is only 	filtered by Sport Type on the database. Searches can be both First and Last name sensitive. The names of matching 	coaches are displayed along with thumbnail profile pics with a REQUEST TO CONNECT button beside each one. That 	will put the Affiliation into a pending mode until it is accepted by the Coach.
	- At the Coach Account end, they will receive a pop-up notification that "[CLUB NAME] has requested for you to be 	a Club Affiliate. Are you are currently coaching at this club? YES / NO." If they select YES, they are added to 	the Club's Coach Affiliates list and will also be able to start posting Players Looking & Trials Info advert types 	on the Club's behalf. These adverts also get populated under the Club's current adverts page. Both the Club & 	Coach (author) are able to edit/delete these as they choose. The only difference with these advert types is 	the Coach Name gets added under a Team Coach field (and cannot be edited by the coach or club). Essentially 	any Coach affiliated account posting an advert is classed as that Club's owned advert.
	If the coach selects NO, the club will see the rejection listed under their Coach Affiliates list with an OK 	button to remove the Coach off their list. The Club can search the Coach again but cannot request another 	connection until 24 hours since rejection has past. The connect button is greyed out with red button text stating 	what date & time the button will become available. After 3 rejections, the connection status between the two 	accounts are blocked that only the admin can refresh at the back end.
	- If a player requests to connect privately with an advert type that is Coach (Author) Club Affiliated, the 	request is sent to both the Club & Coach. Once accepted by either, both the Club and the Coach are added to the 	chat room. Both the Club and Coach are able to use the Broadcast message feature if 2 or more chats of thr same 	advert 	are open.
